Usage: TypeWriter <Command> [Command Arguments]
 
Available Commands:
    TypeWriter Build -i/--input=./src/ -o/--output=./   (Build/compile a src folder)
    TypeWriter Execute -i/--input=./src/                (Execute a compiled project)
    TypeWriter ExecuteBuild -i/--input=./src/           (Build a src folder and Execute it)
    TypeWriter Help                                     (Show this message)
    TypeWriter Init -o/--output=./                      (Start a new project)
